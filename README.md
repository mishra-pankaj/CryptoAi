🪙 CryptoAI: AI-Powered Crypto Prediction Platform

CryptoAI is a full-stack digital asset analysis tool designed to bridge the gap between volatile cryptocurrency data and actionable trading intelligence. By combining real-time market metrics with Large Language Models (LLM), it provides users with automated, high-confidence investment signals.

📺 Project Demo


🚀 Key Features
Real-time Market Intelligence: Integrated CoinMarketCap API to fetch live price action, market capitalization, and 24-hour volume metrics.

Neural Analysis Engine: Leveraged Gemini AI to process complex market data and generate automated "Buy/Sell/Hold" recommendations with a specific confidence percentage.

Interactive Data Visualization: Features a dynamic 30-Day Price History chart and a Sentiment Score gauge for intuitive technical assessment.

User Session Management: Implemented a secure Login and Signup flow (mock-authenticated) to provide a personalized dashboard experience.

Responsive Dark UI: Designed a high-performance, mobile-responsive interface optimized for professional traders and crypto enthusiasts.

🛠️ Tech Stack
Backend: Node.js, Express.js

Frontend: HTML5, CSS3, JavaScript (ES6+)

AI Integration: Google Gemini AI API

Data Source: CoinMarketCap Professional API

Styling: Custom CSS (Flexbox/Grid) for the "Cyber-Dark" aesthetic
